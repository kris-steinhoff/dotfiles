# dotfiles

Personal dotfiles managed with [chezmoi](https://chezmoi.io/).

## Quick start

```bash
chezmoi init kris-steinhoff/dotfiles && chezmoi apply
```

## Update configs

Edit a file on the machine, then capture the change:

```bash
chezmoi edit $FILE
```

See https://www.chezmoi.io/user-guide/daily-operations/ for more details.
