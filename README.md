# Personal Dotfiles

## Setup

On a new machine:

```bash
brew install yadm
yadm clone https://github.com/kris-steinhoff/dotfiles.git
```

## Features

**Shell (zsh)**
- Emacs keybindings with escape → vi mode
- Syntax highlighting & autosuggestions
- Starship prompt
- Git, Docker, AWS aliases & functions

**Git**
- Formatted log (`lg`)
- Interactive rebase helpers (`grbi`, `grbom`)
- Auto-prune orphaned branches (`trim`)

**vim**
- Minimal config with syntax highlighting, 4-space tabs
